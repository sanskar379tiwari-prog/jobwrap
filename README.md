# jobwrap
jobwrap is a free, AI-powered web app that gives brutally honest, structured career roadmaps for tech roles.
Users enter a tech job role (e.g. Backend Developer, Frontend Developer), and the app generates:

Realistic role expectations

Prioritized skills (what actually matters)

Time-bound learning plans

Projects that signal real job readiness

Common traps beginners fall into

Salary reality for the Indian job market (if data avalible)

Built with a Node.js + Express backend, vanilla HTML/CSS/JS frontend, and free text only AI models (Hugging Face).
The focus is on clarity, specificity, and realism  not motivational fluff or guaranteed outcomes.

This project is intended as a learning-focused, open-source experiment in combining prompt engineering, backend validation, and honest career guidance.
